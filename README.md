# Nodejs Project ECE

## Introduction

Our Nodejs project

## Installing

```bash
git clone https://github.com/JProuvost/Node-Project.git
cd Node-Project
npm install
```

## Build

```bash
npm run build
```

## Development

```bash
npm run dev
```

## Database population

```bash
npm run populate
```

If there is any problem with this command, try:
```bash
npm install nodemon
```
